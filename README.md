## Getting Started

1. Clone the DAO Hub repository:

```bash
git clone https://github.com/gxnaitik/DAOHub
```

2. Goto the Frontend directory of the project

```bash
cd DAO Hub/frontend/
```

3. Install Dependencies:

```bash
yarn
```

4. Run the project at `localhost:3000`:

```bash
yarn run dev
```
